## \:dart: About

**THP Final Project - Fullstack Certification - March 2025**

As we were reaching the end of the Fullstack training program with The Hacking Project, we were asked to create a complete web application.

**Nicolas** came up with the idea of developing a modern and user-friendly website for his local Turkish snack based in **Forbach**, in the north of France, named **Le Plaisir**.

The platform allows customers to:

* Browse the menu,
* Customize their dishes (e.g. remove allergens),
* Add items to their cart,
* Proceed to checkout and payment.

On the **Admin side**, managers can:

* View and track orders in real-time,
* Access sales reports and forecasts,
* Count the number of customers,
* Identify recurring clients.

## \:checkered\_flag: Starting

```bash
# Clone this project
$ git clone git@github.com:Guggs57/Le_Plaisir_snack.git

# Access project folder
$ cd le_plaisir_snack

# Install Ruby gems
$ bundle install

# Install JS packages
$ yarn install

# Set up the database
$ rails db:create db:migrate db:seed

# Start the server
$ rails server

# Visit on your browser
http://localhost:3000
```
